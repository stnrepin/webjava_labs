\section{Цель работы}

Знакомство с процессом создания GWT-приложения в среде Eclipce и Maven.

\section{Выполнение работы}

Был разработан проект, по функциональности аналогичный такому из лабораторной
работы №3: пользователь вводит число -- максимальную цену за штуку -- и ему
выводится список лекарств в аптеке с учетом данного фильтра. Приложение
работает на основе фреймворка GWT.

Проект разделен на три части: клиентская сторона, серверная сторона и общий
код. На клиентской сторону описывает HTML-страница, представляющаяся
пользователю, интерфейс RPC-сервис, описывающий серверную часть приложения и
модуль, контролирующий пользовательский интерфейс и соединяющий его с серверной
частью. На серверной части реалзиуется RPC-сервис, передающий некоторые данные
(список лекарств, удовлетворяющий критерию фильтра) клиентской части. В общей
части расположен код валидации вводимых данных и DTO-класс лекартства.
